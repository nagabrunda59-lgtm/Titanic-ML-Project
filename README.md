# Titanic ML Project

## Overview
Machine Learning model to predict Titanic passenger survival 
using 3 different algorithms.

## Tools Used
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn

## Dataset
Titanic Dataset - Kaggle

## Models Used
- Logistic Regression — 79.89%
- Random Forest — 82.68% (Best Model)
- KNN — 69.83%

## Key Findings
- Random Forest performed best with 82.68% accuracy
- Gender and Passenger Class were most important features
- Females and 1st class passengers had higher survival rates
